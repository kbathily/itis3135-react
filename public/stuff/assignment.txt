Summary:

In this assignment, we are going to create a form that will allow visitors to create their own Introduction Website. In a form like page, you are going to request information like personal background, academic background, and much more! From there, your website visitor should be able to add all of their information, click a submit button, and their own introduction page will pop up. Refer back to your introduction page while you test this.

Evaluations:

To test this we will first follow your link, and click the submit button. We will open your existing introduction page in another tab, and compare these two pages. They should be EXACTLY the same except for the H2 (Which should say Introduction Form). This includes spacing, colors, fonts, etc. We should not be able to distinguish the your form result page from your original introduction apart from the additional word in the H2.

Next, we will go in and edit the prefilled values to make sure that the new values show up on the submit page, including a different image. Then we will clear the form and test that. Then test that optional fields can be blank and that the rest are required.

We will try adding and removing courses to the form, and making sure these process as expected.

Prefill info w/ YOUR information using the value attribute, on each field with the value on your introduction.

Insert a 'Submit' button: <button type="submit">Submit</Button>.
